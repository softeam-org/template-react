## Sobre o template

O template para aplicações com Reactjs é criado e mantido por desenvolvedores da softeam com o intuito de organizar e melhorar a produtividade dos projetos feitos pela empresa.

# :construction_worker: Executando

```bash
# Clone o Repositório
$ git clone https://github.com/softeam-org/react-template.git
```

```bash
# Baixe as dependendências
$ yarn install

```

```bash
# Execute
$ yarn start
```
